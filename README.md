# 星卡指挥营

面向低年级孩子的本地离线综合 App：先完成英语和数学训练，获得星星、补给点和补给箱进度，再开包、组卡并打一局儿童策略卡牌演练。

## 产品范围

- 基地：展示今日训练、成长循环、补给箱状态、当前卡组预览和进入战场入口。
- 训练：英语听音/找词/补字母，数学自适应题，答错不扣分、不羞辱、不自动推进。
- 卡库：补给箱、收藏册、当前卡组三个 Tab；收藏册为两列卡牌网格。
- 卡牌：`web/content/cards.js` 提供 96 张原创卡牌、6 套玩家默认卡组、3 套 AI 卡组。
- 战场：单机人机，包含总部生命、行动点、支援区、前线区、手牌、牌库、单位推进和攻击。

## 边界

- 不登录、不上传、不接广告、不做充值、不做排行榜、不做联网对战。
- KARDS 和线下卡牌照片只作为玩法氛围/版式/桌面感参考，不复制官方或照片里的卡名、卡图、文案和数据。
- 所有进度只保存在本机 localStorage。

## 本地预览

```powershell
cd web
python -m http.server 5178
```

访问 `http://localhost:5178`。

## Android 打包

```powershell
cd android
.\gradlew.bat assembleDebug
```

APK 会加载 `web/` 作为本地 assets，音频与离线资源随包进入。

## 数据

- 存档 key：`kids-learn-card-camp-state-v1`。
- 保存内容：英语进度、数学档位、星星、补给点、卡包进度、已获得卡牌、当前卡组、对战记录、设置、已解锁卡组、最近获得卡牌。
- 清空进度必须二次确认。
