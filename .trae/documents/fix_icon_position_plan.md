# 修复图标位置和解释🃏4含义

## 问题分析

### 问题1：类型图标挡住了卡片头像
当前类型图标(`.unit-type-icon`)定位在卡片图片的右上角，遮挡了卡片头像的显示。

### 问题2：🃏4的含义
`🃏${s.hand.length}` 在敌方总部显示，表示敌方当前手牌数量。🃏是扑克牌图标，数字表示手牌张数。这个信息对玩家有策略价值（可以知道敌方手牌数量），但图标和数字的组合不够直观。

## 修复方案

### 1. 调整类型图标位置
将类型图标从右上角移到左下角，避免遮挡卡片头像。

### 2. 优化🃏4的显示
- 添加更清晰的文字说明，如"手牌"
- 调整样式使其更美观

## 文件修改

### web/styles.css
- 修改 `.unit-type-icon` 的位置，从右上角移到左下角
- 修改 `.hand-type-icon` 的位置，从右上角移到左下角
- 添加 `.hq-hand` 样式，优化敌方手牌数量显示

### web/app.js
- 修改 `renderHq` 函数，优化敌方手牌数量的显示文本
